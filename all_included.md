# Comprehensive List of FigureYa Modules

This file provides a complete list of all available FigureYa modules. All listed modules are designed for ease of use and can be run directly using GitHub Actions, eliminating the need for local installation of R and R packages. Simply click on a module name below to navigate to its directory.

To make it even more convenient, these folders have been synchronized to another repository [@ying-ge/FigureYa-Actions](https://github.com/ying-ge/FigureYa-Actions), making it easier to fork and use directly. Additionally, any modification to files in this repository will be automatically synchronized to the `FigureYa-Actions` repository, ensuring that both are always up-to-date.
