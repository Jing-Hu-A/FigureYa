## 如何贡献新工具？

1. 在本页面评论或通过 Pull Request 提交你的工具信息，包括：
   - 工具名称
   - 简介
   - 作者（GitHub 用户名或组织）
   - 项目链接
2. 工具需与 FigureYa 有直接关联，或能与 FigureYa 集成使用。
3. 欢迎补充使用说明或截图，让更多用户了解你的工具。

# FigureYa Ecosystem

Welcome to the FigureYa ecosystem page! Here you'll find tools and extensions built by the community based on FigureYa, helping users discover more powerful features and applications. We invite you to recommend, contribute, or expand this list and make the FigureYa community thrive.

## Community Tools List

| Tool Name    | Description                                                 | Author                | Link                                                        |
| ------------ | ----------------------------------------------------------- | --------------------- | ------------------------------------------------------------ |
| FigureYaLLM  | Extension for FigureYa to integrate large language models (LLMs) and AI features. | [xuzhougeng](https://github.com/xuzhougeng) | [GitHub Repository](https://github.com/xuzhougeng/FigureYaLLM) |

*Feel free to add more tools!*

## How to Contribute a New Tool?

1. Comment on this page or submit a Pull Request with your tool information:
   - Tool name
   - Brief description
   - Author (GitHub username or organization)
   - Project link
2. Tools should be directly related to FigureYa, or easily integrated with it.
3. You’re welcome to add usage instructions or screenshots to help users understand your tool.

## Contact & Discussion

- If you have suggestions, questions, or wish to join development discussions, please reach out via Issue or Discussions.

Let’s build the FigureYa ecosystem together—your participation makes a difference!
